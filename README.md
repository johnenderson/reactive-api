# API RestFul com Spring Webflux - Criando uma API de Filmes

API de Filmes de forma totalmente reativa. Usando os conceitos de **Subscribe**, **Flux** e **Mono** dentro do **Webflux do Spring**.

## - Requisitos minimos:
    * Java 11 SDK
    * Maven
